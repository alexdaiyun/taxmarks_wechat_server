发送文本消息

  • POST    sendTextMessage
                请求参数：

                {
                    "touser": "oNGinuMVf5HsHloMbYvPGlN3VzvM",
                    "content": "2这只是一个测试哦",
                    "url": "http://127.0.0.1/wecenter/?/m/question/12",
                    "title": "通知消息"
                }

                提交的数据为JSON格式

                touser: 接收方的openid
                content: 发送的文本内容
                url: 发送的链接，如果无，请设置为空
                title: 消息标题，如果无，请设置为空

                当只有content时，将发送文本消息
                当有url时，将发送图文消息即包含链接
            
                header设置：

                Content-Type: application/json;charset=utf-8

            
                返回结果
                {
                  'status': true,
                  'result': ''
                }
